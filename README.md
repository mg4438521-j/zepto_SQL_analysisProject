# zepto_SQL_analysisProject
Zepto SQL Data Analysis

Analyzed Zepto's product inventory dataset using PostgreSQL to uncover insights on pricing, discounts, stock availability, and category performance.

Key Highlights
Performed data exploration and cleaning
Removed invalid records and standardized pricing data
Identified top discounted products and out-of-stock high-value items
Estimated category-wise revenue and inventory weight
Analyzed discount trends and product value metrics
Tools
PostgreSQL
SQL

This project demonstrates practical SQL skills in data cleaning, querying, aggregation, and business insight generation using real-world e-commerce data.
